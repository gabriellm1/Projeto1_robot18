## Projeto 1 de Robotica Computacional

### Descrição:
Robô Turtlebot3 Burguer programado em ROS na aula de Robótica Computacional 

### Como usar:
Para rodar o programa execute o arquivo Main.py presente na pasta script.
O robô reconhece a cor azul para seguir objetos e foge de objetos detectados com features(case.png), evitando obstáculos com scan e reagindo a impactos e variações na aceleração via IMU.
Para conhecer seu funcionamento, assistir o video em: https://www.youtube.com/watch?v=OaOlwmFPqMA&feature=youtu.be
